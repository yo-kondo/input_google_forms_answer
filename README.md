# input_google_forms_answer

Googleフォーム自動入力URL作成

# AI依頼内容

## 言語

Golang

## 目的

Googleフォームの入力欄に自動で値を入力する。

## 入力

- 設定ファイル（TOML）で設定する。
    - GoogleフォームのURL
    - 質問番号、回答内容、コメントの配列
        - 質問番号
            - 文字列
        - 回答内容
            - 文字列
            - 入力値が`{today}`の場合はシステム日付（yyyy-mm-dd形式）を設定する。
        - コメント
            - 文字列
            - コメントは処理には使わない。質問内容をメモする。

## 処理

処理の終わりにエンコードしたURLと、デコードしたURLをコンソールに出力する。

## 出力

- GoogleフォームのURLにID、入力値を付加したURLをコンソールに出力する。
    - `usp=sf_link`というクエリストリングがついていたら削除する。
    - `usp=pp_url`のクエリストリングを付与する。
    - GoogleフォームのURLに`entry.質問番号=回答内容`の形でクエリストリングを付加する。

## 制約事項

- プログラムのヘッダーにプログラム名（指示書の一番はじめの見出し1）を記載する。
- 処理がわかりにくいところは、適切なコメントを記載する。
- エラーが発生した場合はコンソールに出力する。原因が特定できるメッセージを出力する。
